Jawaban Soal No 1

Cara Kerja dari website serta jelaskan hubungan antara web server , domain dan hosting ?

  • web server adalah perangkat keras atau perangkat lunak yang menyimpan, mengelola, dan menyajikan konten web kepada pengguna.
  • Domain adalah alamat unik yang digunakan untuk mengidentifikasi suatu situs web di internet. Contohnya adalah "www.nama-website.com".
  • Hosting adalah tempat penyimpanan data dan file yang membuat suatu situs web dapat diakses melalui internet. Saat seseorang mengakses situs web, web server menyajikan konten dari hosting.

Hubungan antara Web Server, Domain, dan Hosting:

  • Domain dan Hosting: Domain dan hosting adalah dua komponen yang terpisah tetapi saling terkait. Domain memberikan alamat unik untuk situs web, sementara hosting menyimpan data dan file situs web tersebut.
  • Web Server dan Hosting: Web server adalah bagian dari hosting yang mengelola permintaan HTTP dan memberikan konten situs web kepada pengguna.
  • Web Server, Domain, dan Pengguna: Ketika seseorang mengetikkan alamat domain di browser, DNS akan mengarahkan permintaan ke server hosting yang menyimpan situs web tersebut.
